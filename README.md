## How to install

1. Clone this repository:

```bash
git clone https://github.com/BoyWhoCriedWolf/mc-landing.git
```

2. Go into project

```bash
cd mc-landing
```

3. Install dependencies

```bash
npm install
```

4. Run project

```bash
npm run dev
```
